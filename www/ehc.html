Download EHC

  • The current version of all compiler versions can be downloaded as [tgz, zip] file.
    • New versions appear whenever a stable point in the development has been reached.
    • Requires the UUST library
  • The current version of the documentation can be downloaded as pdf.
    • (For the coming months the documentation reflects work in progress, as such it is incomplete, more disclaimers, etc)

Download & install Ruler

  • The current version of Ruler can be downloaded (independently from EHC) as [tgz] file.
    • Requires the UUST library
    • Installation:
      1. Unpack the distribution in a new directory (the current distribution has several subdirectories on its toplevel)
      2. cd ruler2
      3. Follow the instructions in the README file in that directory
